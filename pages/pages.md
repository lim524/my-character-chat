# pages 폴더 기능 설명

Next.js 라우팅을 정의하는 페이지 파일들이 모여 있습니다.

## 주요 페이지

### `index.tsx`
- 애플리케이션의 진입점입니다. 홈 화면이나 생성 화면으로의 리다이렉션을 처리합니다.

### `settings.tsx`
- 전역 설정 페이지로, API 키 등록 및 UI 테마 변경 등이 가능합니다.

## 하위 디렉토리

### `api/`
- 서버 사이드 API 엔드포인트들을 포함합니다.

### `characters/`
- 등록된 캐릭터 목록을 보고 관리하는 페이지입니다.

### `chat/`
- 메인 채팅 인터페이스 페이지입니다.

### `create/`
- 새로운 캐릭터를 생성하거나 기존 캐릭터를 가져오는 마법사 페이지입니다.

### `home/`
- 애플리케이션의 대시보드 또는 랜딩 페이지입니다.

### `mypage/`
- 사용자 데이터 백업, 복구 및 로컬 저장소 관리 페이지입니다.
