# My Character Chat (나의 캐릭터 채팅)

나만의 캐릭터를 생성하고, 고도의 AI와 함께 비주얼 노벨 스타일의 대화를 즐길 수 있는 웹 애플리케이션입니다. 캐릭터 생성부터 채팅까지의 모든 과정을 자유롭게 커스텀할 수 있습니다.

## 🚀 주요 기능 및 워크플로우

### 1. 캐릭터 생성 및 관리 (Creation)
캐릭터의 외형, 성격, 세계관을 세밀하게 설정할 수 있습니다.

*   **캐릭터 프로필 설정**: 이름, 설명, 성격, 상황 등을 정의하여 AI의 페르소나를 구축합니다.
*   **다양한 임포트(Import) 지원**: 
    *   기존 캐릭터 카드(PNG, JSON, .charx)를 불러와 즉시 사용할 수 있습니다.
    *   SillyTavern, V2 등 표준 포맷과의 호환성을 제공합니다.
*   **로어북(Lorebook) 관리**: 특정 키워드에 반응하여 AI가 지식을 불러오는 세계관 설정(Lore)을 등록할 수 있습니다.
*   **자산(Asset) 시스템**: 
    *   캐릭터의 다양한 표정 이미지와 배경 이미지를 등록합니다.
    *   채팅 중 AI가 감정에 맞춰 이미지를 변경하거나, 사용자가 수동으로 배경을 전환할 수 있습니다.
*   **추가 인터페이스(Extra Interface) 설계**: 
    *   JSON 규칙을 사용하여 화면에 인벤토리, 장식 오버레이, 커스텀 버튼 등을 직접 배치할 수 있습니다.

### 2. 고도의 채팅 환경 (Chatting)
생성한 캐릭터와 실시간으로 대화하며 몰입감 있는 경험을 제공합니다.

*   **하이브리드 인터페이스**: 
    *   전통적인 채팅 로그 방식과 미려한 **비주얼 노벨(VN) 스타일** 오버레이를 동시에 지원합니다.
    *   긴 응답은 페이지 단위로 나누어 비주얼 노벨처럼 넘겨볼 수 있습니다.
*   **지능형 메시지 파서(Message Parser)**: 
    *   AI 응답 내의 특수 태그를 해석하여 자동으로 배경을 바꾸거나, 캐릭터 스프라이트를 변경하고, 스탯을 업데이트합니다.
*   **프롬프트 커스터마이징**: 
    *   시스템 프롬프트, 캐릭터 프롬프트, 탈옥(Jailbreak) 프롬프트를 세부적으로 조정하여 AI의 답변 스타일을 제어합니다.
*   **세션 관리**: 채팅 내역을 로컬에 저장하고 언제든 다시 불러올 수 있습니다.

### 3. 기술적 세부 기능 (Advanced)
*   **정규식 스크립트(Regex Scripts)**: 입출력 텍스트를 정규식으로 자동 치환하여 고유의 어투나 시스템 명령을 처리합니다.
*   **멀티 모델 지원**: Google Gemini, Anthropic Claude 등 다양한 AI 모델과 연동할 수 있습니다.
*   **로컬 퍼스트(Local-First)**: 모든 캐릭터 데이터와 채팅 내역은 브라우저의 IndexedDB 및 LocalStorage에 안전하게 저장됩니다.

## 🛠️ 시작하기

1.  **캐릭터 생성**: 사이드바의 '생성' 메뉴에서 새로운 캐릭터를 만들거나 기존 카드를 임포트하세요.
2.  **설정**: 마이페이지에서 사용할 AI 모델의 API 키를 입력합니다.
3.  **채팅 시작**: 홈 화면이나 캐릭터 목록에서 원하는 캐릭터를 클릭하여 대화를 시작하세요!
