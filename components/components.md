# components 폴더 기능 설명

애플리케이션 전반에서 사용되는 공통 UI 컴포넌트들입니다.

## 주요 컴포넌트

### `BottomNav.tsx`
- 화면 하단 네비게이션 바입니다. (현재 코드상에서 비활성화 상태)

### `CharacterProfileModal.tsx`
- 캐릭터의 이름, 설명, 성격 등을 확인하고 수정할 수 있는 모달입니다.

### `ChatMenu.tsx`
- 채팅과 관련된 다양한 액션(로그 보기, 설정 등)을 담은 메뉴 컴포넌트입니다.

### `CreateStepTab.tsx`
- 캐릭터 생성 위저드에서 각 단계를 표시하는 탭 컴포넌트입니다.

### `DatingSimScreenPreview.tsx`
- 비주얼 노벨 또는 연애 시뮬레이션 스타일의 화면 레이아웃 미리보기를 제공합니다.

### `ExtraInterfaceOverlay.tsx`
- AI가 제어하는 커스텀 UI 요소(버튼, 이미지 등)를 렌더링하는 오버레이 레이어입니다.

### `ImageUploader.tsx`
- 이미지 업로드 및 처리를 위한 유틸리티 컴포넌트입니다.

### `MessageParser.tsx`
- AI 응답에서 특수 태그(인터페이스 명령, 자산 변경 등)를 해석하고 마크다운을 HTML로 변환합니다.

### `SideMenu.tsx`
- 전역 네비게이션을 위한 사이드바 메뉴입니다.

### `TopNav.tsx`
- 상단 네비게이션 바입니다.
